# ogr2ogr でデータ追加（PowerShell やコマンドプロンプトで実行）

- postgis 直下に migration フォルダを作成

```
docker run --rm -v "C:/Users/keita/Desktop/programing/nextjs/OpenDataChallenge/03_プロトタイプ/postgis/migration:/data" osgeo/gdal:alpine-small-3.6.3 ogr2ogr -f "PostgreSQL" PG:"host=host.docker.internal user=docker dbname=postgres password=docker port=5432" /data/N05-23_Station2.geojson -nln public.n05_23_station2 -append


```

```
ogr2ogr -f "PostgreSQL" PG:"host=localhost user=docker dbname=postgres password=docker" /tmp/migration/N05-23_Station2.geojson -nln public.n05_23_station2 -append
```
