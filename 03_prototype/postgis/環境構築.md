# docker 立ち上げコマンド

```
docker compose up -d
```

# docker 停止コマンド

```
docker compose down -v
```

# pipeline.py を実行して初期データを postgres に入れる

```
python pipeline.py
```

# postgres に入るコマンド

```
docker exec -it postgis_container psql -U docker -d postgres
```
