## 環境構築

- 用意するもの

  - vscode

  - node 環境(バージョンは何でもよいが、node -v コマンドが実行できることを確認する)
    ```
    node -v
    v20.12.1
    ```
  - yarn を利用できることを確認
    ```
    yarn --version
    1.22.22
    ```
  - nextjs のプロジェクト作成
    ```
    // test-app-name には好みの名前を設定してください
    npx create-next-app test-app-name
    ```
